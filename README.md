# API REST de Películas

API REST desarrollada con Node.js y Express siguiendo el patrón MVC.
Permite realizar operaciones CRUD sobre un catálogo de películas.

## Tecnologías
- Node.js
- Express
- JavaScript (ES Modules)
- Arquitectura MVC

## Endpoints principales
- GET /movies
- GET /movies/:id
- POST /movies
- PATCH /movies/:id
- DELETE /movies/:id

## Deploy
La API se encuentra deployada en Render:
https://rest-api-deploy-w3sz.onrender.com

## Nota
Proyecto educativo orientado al aprendizaje de APIs REST con Node.js.
La persistencia se realiza en memoria. Es una API educativa sin persistencia.
Los datos pueden reiniciarse ante redeploys o inactividad del servicio.
